Generador de Contraseñas

Instrucciones y recomendaciones

Para inicializar el Generador de contraseñas utilizar la Funcion generadorDeContraseñas() en la consola.

Le pedira ingresar unos valores a travez de pops ups

El primer valor a ingresar es la cantidad de digitos de la conseña a generar.

Le dejo una gia para el sig valor.

  • Ingrese 1 para contraseña numerica.
  • Ingrese 2 para contraseña alfanumerica.
  • Ingrese 3 para contraseña alfanumerica y simbolos.
